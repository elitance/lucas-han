Intro

I started my coding-life when I was 12, with command prompt script. It was nice. It had a function-similar stuff. As I grow up, I started to want more than just command prompt. I knew what I was doing was not productive, creative, and useful. It was really useless program that had no special purpose. I started my really coding from 13 years old. My first language was Python, which famous for being introductory. Learning Python, I felt no difficulty. I tried HTML and markdown next. It was even easier than Python!

I tried C, CSS, JavaScript, Swift, and MySQL. All of them was not so hard. One by one, I started cloning the apps that I always used. For instance, todo app, searching app, painting app, and code editor (thanks to Nomad Coders). After trying such things, my love toward coding became bigger and bigger. And finally, I end up dreaming becoming a programmer, especially web full-stack developer.
